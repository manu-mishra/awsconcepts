          <h3><a href="https://aws.amazon.com/blogs/aws/getting_top_sel/">Getting Top Sellers for a Browse Node</a></h3>
          <p>The most recent release of ECS adds a very powerful new feature the ability to retrieve the best selling items associated with a particular browse node. Here is a sample call: http://xml.amazon.com/onca/xml?Service=AWSECommerceService&amp;Version=2005-07-26&amp;Operation=BrowseNodeLookup&amp;SubscriptionId=[YOUR_SUBSCRIPTION_ID]&amp;BrowseNodeId=1000&amp;ResponseGroup=TopSellers You will need to insert your Subscription ID into this URL in order to see the results.</p>
