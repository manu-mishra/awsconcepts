          <h3><a href="https://aws.amazon.com/blogs/aws/soapui_web_serv/">SOAPUI Web Service Inspection and Testing Tool</a></h3>
          <p>Soapui is an open source Java 1.5 tool (with a Swing UI) for inspecting and testing SOAP-based web services. It supports WSDL importing, request generation, an unlimited number of requests per operation, multiple service endpoints for the same service, validation, three types of authentication, and background processing. (found on webservices.org)</p>
